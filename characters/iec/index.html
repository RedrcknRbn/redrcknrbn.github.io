This universe is used for all of my main "stories", where everything is a connected universe. This universe is similar to that of an scp universe, where there's supernatural creatures and beings, except the [[Institution of Entity Containment]] (effectively this universe's SCP Foundation) only focuses on containing primarily dangerous entities. This will give me enough leniency to create unique stories. This is also set slightly in the future, technology wise (2029-2030?) so that way I can play around with more sci-fi like tropes, whilst still being grounded in reality. 

I want a major event to happen canonically that'll set off world-wide events. An idea I had was to have [[IEC-294 (The Vending Machine)]] cause the [[Catalyst Event]], where someone orders something like Anti-Matter that completely destroys the main containment facility, freeing entities, whilst also sending shockwaves all around the world. This will also help me set up some problems that other stories' characters have to solve, like [[IEC-448 (The Darmask)]]'s sudden shift, or [[Freerun Facility]] going dark.
