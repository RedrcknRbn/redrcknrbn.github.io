Discord Markup Tutorial

Written on June 6th, 2023

This blog will go into detail on what Discord's Markdown is, and how to use it.

To use Bold Text, (Equivalent to using <b>), put an *asterisk around your text.

*Example* to Example

To use Italicized Text, (Equivalent to using <i>), put two **asterisks around your text.

**Example** to Example

Interestingly, Italicized Text can also be created using two __Underscores

__Example__ to Example

To use Both, (Equivalent to using <b><i>), put three ***asterisks around your text.

***Example*** to Example

To use Underline Text, (Equivalent to using <u>), put an _underscore around your text.

_Example_ to Example

To use Italicized Underlined Text, (Equivalent to using <u><i>), put three ___Underlines around your text.

___Example___ to Example

You can also Mix and Match, like ***___Hi___*** to Hi

While I can't demonstrate headers here well, I can tell you how to use them!

Use #hashtag to create a "Header Text", with the amount of hashtags determining how big it is.

In HTML, we use H1 for the biggest header, H2 for a smaller header, H3 for an even smaller header. The same thing applies to the amount of Hashtags you use.

One hashtag is the biggest, 2 is a smaller header, and 3 is the smallest.

Keep in mind you DO have to put a space after the hashtag(s)

# Hey!
