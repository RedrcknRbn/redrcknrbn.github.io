The Joys of Obsidian

Written on April 29th, 2025

I'm gonna use this time to rant about Obsidian, the note taking app, so be warned, incoherent rant incoming.

OH MY GOD OBSIDIAN IS SO GOOD

Ok, so, for reference, I used to use Samsung Notes for taking notes of things, but now I use Obsidian. Obsidian is a FOSS (Free Open Source Software) notes taking app, which primarily uses MD (Markdown) to take notes in.

The difference is night and day compared to the two apps.

Because Obsidian uses Markdown, it's extremely easy to actually write and format notes in the way you prefer

Also, you can set Obsidian up to automatically create daily notes, using it as a journal. I have a seperate folder for my Daily Notes, I've gotten into the habit of writing daily notes. This is useful for me, because I am extremely forgetful, and have memory loss. When you're constantly writing things down, you forget less and less! Also, journaling is just good if you are doing therapy, as you can easily write down your feelings in the moment, then tell the therapist about them.

Also, the ability to create links between two notes is super useful! I can connect my notes super easily, then jump between topics seamlessly.

As a side note, Obsidian has this "Graph View" feature, where it shows all of the connections between notes that I've made in an easy to view graph. I've set up my graph view to assemble itself like a constellation, and I find it interesting to see how it grows over time!

Cool, right!

Also, you can sort notes into easy to use folders. This comes in handy for when you write about many topics, and you wanna sort the topics.

Theres also this interesting canvas feature, where you can connect notes, images, and topics in a canvas, similar to visual coding/material creation.

I use a program on my Phone and PC called Syncthing, which allows me to sync my Obsidian Vault between my phone and pc completely for free! I can write stuff down on the go, and access it from home.


That's all I wanted to talk about. Use Obsidian, genuinely, it's so good.
